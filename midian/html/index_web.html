This program lets you import Midi-format files into Synthstrom Deluge SONG.XML files.

To use this program, press the "Choose File" button and select a Midi file to work with. You will then see one or more Midi tracks displayed graphically. To add a track, click on the "+ Song" button and the track will be converted and appended.

Song File:
  • Download the updated song by clicking on the Download button at the bottom.
  • To add to an existing song, open it with the "Choose File" button.
  • Select a subset of a track to extract by clicking and dragging.
  • The "→ Clip" button puts the midi track data on the clipboard. You can then paste it into the main XML viewer program.
  • At present, the program only converts Note On/Note Off events.
  • The Midi file you choose and the SONG.XML file are processed entirely on your own computer. No information about it is sent anywhere else.
